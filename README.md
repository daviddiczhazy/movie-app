# movie-app

## Project setup
```
npm install
```

### Instal jsno server for mock database
```
npm install json-server -g
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Starts a Json server for mock database
```
json-server db.json
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
